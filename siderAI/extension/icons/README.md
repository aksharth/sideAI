# Extension Icons

Place your extension icons here with the following names:
- `icon16.png` (16x16 pixels)
- `icon48.png` (48x48 pixels)
- `icon128.png` (128x128 pixels)

You can create icons using:
- Online icon generators
- Design tools like Figma, Canva, or Adobe Illustrator
- Or use simple colored squares as placeholders

The icons should represent your SiderAI extension brand.
